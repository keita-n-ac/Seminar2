```python

print((18 + 6) / 3) # 24 / 3
```

##### Q3
- プログラム
```python
print(25 / 4)  # 除算結果（小数）
print(25 // 4) # 切り捨て除算結果（整数）
print(25 % 4)  # 剰余結果
```
- 出力
  - ``6.25``
  - ``6``
  - ``1``
