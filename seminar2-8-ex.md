# Pythonライブラリ基礎 練習問題20選

このドリルは、標準ライブラリ（math, random, calendar）、NumPy、Matplotlibの基礎的な使い方を確認するための練習問題です。
内容は以下のURL（ https://github.com/keita-n-ac/ml-ex/blob/main/lec8.ipynb ）となります．

## 第1部：標準ライブラリ編 (math, random, calendar)

### 問1: mathライブラリの基礎
`math`ライブラリをインポートし、3の4乗と、144の平方根（ルート）をそれぞれ計算して出力してください。

### 問2: mathライブラリの別名
`math`ライブラリを`m`という別名（エイリアス）でインポートし、`m.pow`を使って10の2乗を計算して出力してください。

### 問3: randomライブラリ（整数）
`random`ライブラリを使用して、10以上20以下のランダムな整数を1つ出力してください。

### 問4: randomライブラリ（リストからの選択）
次のリスト `fruits = ['apple', 'banana', 'cherry', 'grape']` から、ランダムに1つの要素を選んで出力するプログラムを作成してください。

### 問5: randomライブラリ（繰り返し）
`for`文と`random`ライブラリを使用して、1以上100以下のランダムな整数を5回連続で出力してください。

### 問6: calendarライブラリ
`calendar`ライブラリを使用して、2025年1月のカレンダーを出力してください。

## 第2部：NumPy配列の作成と基礎

### 問7: リストからNumPy配列への変換
リスト `[10, 20, 30, 40, 50]` を作成し、それを`numpy`の配列（ndarray）に変換して出力してください。また、その型（type）も出力してください。

### 問8: 2次元配列の作成
次の2重リスト `[[1, 2, 3], [4, 5, 6], [7, 8, 9]]` を`numpy`配列に変換して出力してください。

### 問9: arangeの使用 (昇順)
`np.arange`を使用して、0から9までの整数（0, 1, ..., 9）が格納された配列を作成し、出力してください。

### 問10: arangeの使用 (ステップと降順)
`np.arange`を使用して、30から開始して0まで（0は含まない）、-3ずつ減っていく配列（30, 27, ..., 3）を作成して出力してください。

### 問11: linspaceの使用
`np.linspace`を使用して、0から100までの範囲を、等間隔に5分割した配列を作成して出力してください。

## 第3部：NumPy配列の演算と操作

### 問12: ベクトルへのスカラー演算
配列 `a = np.array([1, 2, 3, 4, 5])` を作成し、すべての要素を10倍にした配列を出力してください。

### 問13: ベクトル同士の演算
2つの配列 `a = np.array([10, 20, 30])` と `b = np.array([1, 2, 3])` を作成し、`a` から `b` を引いた結果（引き算）を出力してください。

### 問14: 要素一括操作（平方根）
配列 `a = np.array([4, 9, 16, 25])` に対して、`np.sqrt`を使用してすべての要素の平方根を求めた配列を出力してください。

### 問15: 要素一括操作（絶対値）
配列 `a = np.array([-5, -2, 0, 3, 8])` に対して、`np.abs`を使用してすべての要素を絶対値にした配列を出力してください。

## 第4部：スライスとグラフ描画 (Matplotlib)

### 問16: スライスによる抽出
配列 `data = np.arange(20)` を作成し、スライス記法を使って「インデックス5から10の手前まで（5, 6, 7, 8, 9）」を抽出して出力してください。

### 問17: スライスによる書き換え
配列 `data = np.zeros(10)` （すべて0の要素数10の配列）を作成し、スライスを使ってインデックス3から6の範囲に `99` を代入し、結果を出力してください。

### 問18: 基本的なグラフ描画
`matplotlib.pyplot` と `numpy` を使用します。
x軸を `np.arange(0, 10, 0.1)` とし、`y = 2x` のグラフ（直線）を描画し、表示してください。

### 問19: ラベル付きグラフ
問18と同様にx軸を定義し、今度は `y = x^2` （2次関数）のグラフを描画してください。その際、x軸のラベルに「Time」、y軸のラベルに「Distance」という名前を付けて表示してください。

### 問20: 複数のグラフの描画
x軸を `np.linspace(-5, 5, 100)` とします。
この範囲で、`y1 = x^2` と `y2 = x^3` の2つのグラフを計算し、**1つの図の中に重ねて**（`plt.plot`を2回続けて実行して）表示してください。
